1. What You Need Before Starting

Before diving in, make sure you have:

  • A working Debian-based distribution (Debian, Ubuntu, Linux Mint, Pop!_OS, or similar)
  • An internet connection
  • Root access or sudo privileges
  • At least 20GB free space for games (more if you play larger titles)
  • A graphics card that supports Vulkan (most do from the last 8-10 years)

Open a terminal. You will use it for most of the installation steps. Copy and paste the commands, and press Enter after each one. If something asks for your password, type it in (you will not see characters as you type, that is normal).

2. Enabling 32-bit Architecture

Many games, especially older ones and Windows games running through compatibility layers, need 32-bit libraries. Most Debian-based systems ship with 64-bit by default, so you need to add 32-bit support if it is not already enabled.

Run this command:

sudo dpkg --add-architecture i386

Then update your package list:

sudo apt update

That is it. Your system can now install both 64-bit and 32-bit packages.

3. Installing Graphics Drivers

Proper graphics drivers are the most important part. Here is how to set them up based on your hardware.

NVIDIA

NVIDIA requires proprietary drivers. On Debian-based systems, this is straightforward.

First, enable the non-free and contrib repositories if you are on Debian. Edit your sources list:

sudo nano /etc/apt/sources.list

Find lines that say main and add contrib non-free after them. They should look like:

deb http://deb.debian.org/debian/ bookworm main contrib non-free
deb http://deb.debian.org/debian/ bookworm-updates main contrib non-free
deb http://security.debian.org/debian-security bookworm-security main contrib non-free

If you are on Ubuntu or a derivative, the drivers are in the default repositories. Install the recommended driver:

sudo apt update
sudo apt install nvidia-driver firmware-misc-nonfree

On Ubuntu, you can also use the proprietary GPU driver PPA for newer drivers:

sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt update
sudo apt install nvidia-driver-550

Reboot when finished:

sudo reboot

AMD / Radeon

AMD cards work with open-source drivers that come with most Debian-based distributions. They are already there, but you may want newer versions for better gaming performance. For Vulkan support (needed for Proton and many games), install:

sudo apt update
sudo apt install mesa-vulkan-drivers mesa-vulkan-drivers:i386

If you are on Ubuntu or a derivative, the Kisak or Oibaf PPAs offer newer Mesa drivers:

sudo add-apt-repository ppa:kisak/kisak-mesa
sudo apt update
sudo apt upgrade

On Debian, you can enable backports for newer drivers:

echo "deb http://deb.debian.org/debian bookworm-backports main contrib non-free" | sudo tee /etc/apt/sources.list.d/backports.list
sudo apt update
sudo apt install -t bookworm-backports mesa-vulkan-drivers mesa-vulkan-drivers:i386

Intel Integrated Graphics

Intel drivers are open-source and included. Just install Vulkan support for better compatibility:

sudo apt update
sudo apt install mesa-vulkan-drivers mesa-vulkan-drivers:i386

That is all you need.

4. Installing Steam

Steam is the easiest way to get thousands of games. On Debian, enable non-free if you have not already, then:

sudo apt update
sudo apt install steam-installer

On Ubuntu and derivatives, Steam is in the multiverse repository:

sudo add-apt-repository multiverse
sudo apt update
sudo apt install steam

After installation, launch Steam from your application menu. The first launch will download updates and set up the Steam runtime. Log in with your Steam account and start downloading games.

5. Understanding Proton

Proton is a tool built into Steam that lets you run Windows games on Linux. Most games that are not native to Linux work through Proton. Here is how to enable it:

In Steam, go to Steam > Settings > Compatibility. Check "Enable Steam Play for all other titles". Steam will restart and now nearly every game in your library can be installed and played.

Some games work better with specific Proton versions. You can change this per game by right-clicking the game, selecting Properties > Compatibility, and choosing a different Proton version. Proton Experimental and Proton GE (a community version) often have the latest fixes.

To install Proton GE:

wget https://github.com/GloriousEggroll/proton-ge-custom/releases/latest
# Then extract the downloaded file to ~/.steam/root/compatibilitytools.d/
# Create the folder if it does not exist

Start with the built-in Proton versions first. They work for most games.

6. Lutris Game Launcher

Lutris handles games from many sources: GOG, Epic Games Store, Humble Bundle, and standalone installers. It manages Wine versions, runners, and settings for you.

Install Lutris:

sudo apt update
sudo apt install lutris

On Ubuntu, you may want the official Lutris PPA for the latest version:

sudo add-apt-repository ppa:lutris-team/lutris
sudo apt update
sudo apt install lutris

After installation, visit the Lutris website, find your game, and click the install button. Lutris will handle the rest. It automatically sets up Wine, Proton, or native runners based on what the game needs.

7. Heroic Games Launcher

Heroic is a lightweight launcher for Epic Games Store and GOG games. It is great if you have a library on those platforms.

Install Heroic:

wget -O- https://heroicgameslauncher.com/heroic_amd64.deb
sudo dpkg -i heroic_amd64.deb
sudo apt install -f

Or on Debian-based systems with Flatpak support:

flatpak install flathub com.heroicgameslauncher.hgl

Log in to your Epic or GOG account, and your library appears. Heroic can launch games with Proton or Wine, and it supports cloud saves.

8. Performance Tweaks

A few simple adjustments can make games run better.

Gamemode: A daemon that optimizes system performance while gaming. Install it:

sudo apt install gamemode

Then add gamemoderun %command% to Steam launch options for each game. Lutris and Heroic have Gamemode toggles built in.

Disable compositing: If you use GNOME or KDE, desktop compositing can add input lag. On KDE, press Alt+Shift+F12 to toggle compositing off while gaming. On GNOME, you can install an extension like "Disable Compositor for Fullscreen Windows".

CPU governor: Set your CPU to performance mode:

sudo apt install cpufrequtils
echo 'GOVERNOR="performance"' | sudo tee /etc/default/cpufrequtils
sudo systemctl restart cpufrequtils

Swappiness: Reduce swapping to avoid stutters:

echo "vm.swappiness=10" | sudo tee -a /etc/sysctl.conf
sudo sysctl -p

9. Troubleshooting Common Issues

Game does not launch on Steam: Try a different Proton version. Right-click the game > Properties > Compatibility > Force the use of a specific Steam Play compatibility tool. Proton Experimental or Proton GE often fix launch issues.

No audio in game: Install additional Wine libraries:

sudo apt install wine wine32 wine64 libwine libwine-development

Game runs but has graphical glitches: Update your graphics drivers. For AMD, try the Kisak PPA. For NVIDIA, install the latest driver from the graphics-drivers PPA.

Steam fails to start: Launch from terminal to see error messages:

steam

Missing libraries are the most common cause. Install any reported missing packages with apt.

Gamepad not working: Install Steam Device Input support or check your distribution's gamepad drivers. Most modern controllers work out of the box.

Low FPS compared to Windows: Check if the game is using the correct graphics card (especially on laptops). Run:

DRI_PRIME=1 %command%

in Steam launch options to force discrete GPU usage.

Game crashes with Vulkan errors: Verify Vulkan is installed correctly:

vulkaninfo

If that command fails, reinstall Vulkan drivers for your GPU.
