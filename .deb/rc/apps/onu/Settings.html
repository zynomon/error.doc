Settings Guide

Onu's settings are organized into a clean tabbed interface. This guide walks you through every option, what it does, and how to configure the browser to work the way you want. You can access settings by pressing Ctrl+, or through the menu button in the navigation toolbar.

1. General Settings

This tab controls how the browser behaves at startup and during normal use.

Startup

Home URL: The page that opens when you click the home button or open a new tab. Default is onu://home. You can set any valid URL here.

Restore webpage sessions on startup: When enabled, Onu remembers all open tabs from your last session and restores them the next time you start the browser. This is useful if you want to pick up where you left off.

Browser Behavior

Ask for confirmation when closing the browser: Shows a confirmation dialog when you try to close Onu. If you disable this, the browser closes immediately without asking. The checkbox in the dialog lets you remember your choice for next time.

Enable Developer Mode (Inspect Element): Allows access to the developer tools (F12) and the "Inspect Element" option in the context menu. Useful for web development or debugging.

Content Blocking

Enable AdBlock: Blocks advertisements and tracking domains based on the blocked_hosts.txt file. The file uses a simplified format similar to EasyList. Click Edit blocked_hosts.txt to open the file in your system's default text editor.

Blocked hosts are checked against page requests. If a request matches a blocked domain, it is blocked before loading.

Actions

Edit Keybinds: Opens the keybindings editor where you can customize keyboard shortcuts for all browser actions. See the Keybinds Editor section for details.

Open Config Folder: Opens the configuration directory in your file manager. This is where all settings, themes, extensions, and custom files are stored.

Clear Icon Cache: Deletes all cached icons (favorites, history, themes) and regenerates them when needed. This can fix display issues with icons not showing correctly.

Remove All Data (Reset): Completely wipes all browser data including history, favorites, settings, themes, and cache. The browser will restart after this operation. This action cannot be undone.

2. Toolbars Settings

This tab controls all toolbars: their visibility, locking behavior, and button settings.

General Toolbar Settings

The main table shows all five toolbars with three toggles each:

  • Visible: Show or hide the toolbar entirely
  • Lock Dragging: Prevents the toolbar from being moved to a different location
  • Lock Floating: Prevents the toolbar from being undocked into a floating window

The Apply Current Settings to All Toolbars button copies the visibility and lock settings from the Navigation toolbar to all other toolbars.

Navigation Toolbar

This toolbar contains back, forward, reload, stop, zoom, volume, and menu buttons.

Button Visibility: Choose which buttons appear:

  • Show Volume Slider
  • Show Reload Button
  • Show Forward Button
  • Show Menu Button

Icon Size: Set the size of navigation icons from 16 to 48 pixels.

Search Toolbar

This toolbar contains the address bar and search engine selector.

Show Search Engine Picker: When enabled, a dropdown appears next to the address bar showing available search engines. When disabled, the address bar works with the default search engine only.

Search Engines: Manage your search engines:

  • Add: Create a new search engine with a name, URL template (use %1 for the search query), and optional icon name
  • Edit: Modify an existing search engine
  • Remove: Delete a search engine
  • Set as Default: Make the selected search engine the default

Suggestion Sources: Control where address bar suggestions come from:

  • Suggest from History: Show URLs and titles from your browsing history
  • Suggest from Favorites: Show your saved favorites
  • Suggest from Remote API: Fetch suggestions from an external API

When Remote API suggestions are enabled, you can configure:

  • API URL: The endpoint that returns suggestions. Use %1 as a placeholder for the search query
  • Parser Script: JavaScript code that processes the API response and returns an array of suggestion strings

Favorites Toolbar

Items Limit: Choose how many favorites appear on the toolbar (3, 5, 10, 15, 20, or 30).

Open Favorites Manager: Opens the full favorites management dialog where you can edit, reorder, or delete favorites.

Recent Toolbar

Items Limit: Choose how many recent items appear on the toolbar.

Clear Recent Items: Deletes all browsing history immediately.

Open History Manager: Opens the full history viewer with search, sorting, and individual deletion options.

Tab Bar

Tab bar settings are managed in the General tab of toolbars. When the tab bar is docked to the left or right side of the window, tabs display vertically with icons and text.

3. Appearance Settings

This tab controls how Onu looks and feels.

Fonts

Font Family: Choose the font used throughout the browser interface. This does not affect webpage fonts.

Font Size: Set the base font size in points (8-24 pt).

Icons and Style

Icon Theme: Select an icon theme from your system. Options include "System Default" and any installed icon themes in /usr/share/icons, ~/.local/share/icons, or ~/.icons.

Qt Style: Choose the underlying widget style. Options include Fusion, Windows, and other styles provided by your Qt installation.

Theme Editor

Onu supports custom QSS themes. The theme editor has two views:

Theme List View: Shows all available themes. Built-in themes (Onu-Dark, Onu-Light, System) cannot be deleted. Custom themes appear here.

  • Create New Theme: Enter a name to create a new blank theme
  • Delete Theme: Remove a custom theme (built-in themes cannot be deleted)

QSS Editor View: When you select a theme from the list, you switch to the editor view. Here you can edit the theme's QSS code directly. The preview updates as you type. Click Save Theme to save your changes, or Back to List to return to the theme selector.

QSS (Qt Style Sheets) uses CSS-like syntax to control every aspect of the interface. For reference, see the Qt documentation or look at the built-in Onu-Dark.qss and Onu-Light.qss files in the themes directory.

4. Privacy and Downloads

Webpage Settings

Enable JavaScript Execution: Turns JavaScript on or off for all pages. Disabling can break many websites but improves security and privacy.

Load Images: Controls whether images load automatically. Disabling can speed up page loads on slow connections.

Delete Cookies in Every Session: When enabled, all cookies are deleted when you close the browser. No persistent cookies are stored.

Send Do Not Track Headers to All Websites: Adds DNT: 1 and Sec-GPC: 1 headers to all requests. This tells websites you prefer not to be tracked.

Advanced

Chromium Flags: Add custom command-line flags passed to the underlying Chromium engine. Separate multiple flags with spaces. Example: --disable-gpu --ignore-certificate-errors. Use with caution.

Encryption

Onu can encrypt sensitive data (history, favorites, sessions, extension preferences, site permissions) with a custom password.

Encryption Method:

  • Default (System-based): Uses a key derived from your system information. Data is encrypted but does not require a password to unlock
  • Custom Password: Uses a password you set. You will be prompted for this password each time you start Onu

Set/Change Encryption Password: Opens a dialog to set or change your custom password. Requirements:

  • Password cannot be empty
  • Both entries must match
  • Minimum length of 4 characters

Important Security Notes:

  • After 10 failed login attempts, Onu will refuse to start
  • To reset after 10 failures, delete encrypted.conf
  • Deleting this file permanently erases all encrypted data (history, favorites, sessions, extension preferences, site permissions)
  • If you switch from Custom to Default encryption, you must verify your password first to decrypt your data

Downloads

Download Path: The folder where downloaded files are saved. Click Browse to select a different location.

Ask for Approval When Download Requested: Shows a confirmation dialog before each download begins.

Allow Duplicate Downloads: Allows downloading the same file multiple times. When disabled, Onu checks if the file already exists and asks before overwriting.

Smart Path (organize downloads by file type): Automatically creates subfolders in your download directory for different file types (images, documents, archives, etc.).

Open Folder When Download Completes: Opens the download folder in your file manager after each download finishes.

Open Downloads Manager: Opens the downloads manager window (also accessible with Ctrl+J).

Site Permissions

Manage Site Permissions: Opens the Site Permissions Manager where you can view and edit permissions for all websites.

This button takes you to a separate dialog with two tabs:

  • Global Defaults: Set default permission behavior for all sites (Allow, Deny, Ask)
  • Sites: View and edit permissions for individual websites that have custom overrides

Permissions include: geolocation, camera, microphone, notifications, popups, clipboard, local fonts, screen sharing, and file access.

5. Scripts and Extensions

Browser Extensions

Onu supports dynamic extensions loaded from ~/.config/Onu/Conf/extensions. Extensions are .so (Linux) or .dll (Windows) files that can modify browser behavior, add toolbars, or interact with web content.

Open Extension Manager: Opens the full extension management dialog where you can view installed extensions, enable/disable them, and see metadata (name, version, maintainer, description).

Open Extensions Folder: Opens the extensions directory in your file manager, where you can add or remove extension files manually.

User Script (JavaScript)

You can inject custom JavaScript code into every page you visit. This is useful for creating browser-wide scripts that modify page behavior or add functionality.

The editor has a monospace font and includes basic controls:

  • New Script: Clear the editor to start fresh
  • Clear Script: Remove all script content

Your script runs on every page load after the DOM is ready. Use this for userscripts, page modifications, or custom automation.

6. Additional Managers

Keybinds Editor

Access this from Settings > General > Edit Keybinds.

The keybinds editor shows a list of all actions and their current keyboard shortcuts. Features:

  • Search: Filter actions by name
  • Edit a keybind: Click the keybind button, then press the desired key combination. The button shows "..." while waiting for input
  • Reset to default: Click the circular arrow button next to a changed keybind to revert it to the default
  • Conflict detection: If you set two actions to the same keybind, a warning appears with options to cancel or restore all defaults

Click Save to apply changes or Cancel to discard.

Favorites Manager

Access this from Settings > Toolbars > Favorites Toolbar > Open Favorites Manager.

The favorites manager lets you organize your saved websites:

  • Search: Filter favorites by title or URL
  • Add New: Create a new favorite with title, URL, and optional custom icon
  • Edit: Modify an existing favorite
  • Remove: Delete selected favorites
  • Clear All: Delete all favorites (with confirmation)
  • Drag and Drop: Reorder favorites by dragging items in the list
  • Right-click menu: Open in new tab, copy URL, edit, delete

History Manager

Access this from Settings > Toolbars > Recent Toolbar > Open History Manager.

The history manager shows your browsing history grouped by date:

  • Search: Filter by title or URL
  • Sort: Sort by recent first, oldest first, or alphabetically
  • Remove Selected: Delete individual history entries
  • Clear All: Delete all history (with confirmation)
  • Right-click menu: Open in new tab, copy URL, add to favorites, remove from history

Double-click an entry to open it in a new tab.

Extension Manager

Access this from Settings > Scripts > Open Extension Manager.

The extension manager displays all discovered extensions with:

  • Search and filter: Filter by name, maintainer, description, or state (All, Enabled, Disabled, Unspecified)
  • Extension details: Click an extension to see its name, maintainer, version, file path, and description
  • State controls: Choose Enable, Disable, or Unspecified for each extension
  • Add Extension: Select .so/.dll files to install new extensions
  • Remove: Delete the extension file from disk
  • Open Folder: Open the extensions directory in your file manager
  • Refresh: Rescan the extensions directory
  • Get extensions: Opens the Onu extensions repository on GitHub

When you first install extensions, you will see a dialog asking you to specify whether to enable, disable, or keep unspecified for each one.

Site Permissions Manager

Access this from Settings > Privacy > Manage Site Permissions.

The Site Permissions Manager has two tabs:

Global Defaults: Set default permission behavior for all websites. Options for each permission type: Allow, Deny, or Ask (prompt each time).

Sites: View and edit permissions for individual websites that have custom overrides:

  • Search: Filter the list of sites
  • Select a site: View all its permission overrides
  • Edit permissions: Change any permission for that site
  • Reset all for this site: Remove all overrides and revert to global defaults

The Clear All Site Permissions button removes all site-specific overrides.

7. Search Bar Integration

The search toolbar integrates all these settings:

  • Address bar: Shows the current page URL and accepts new URLs or search queries
  • Search engine picker: Dropdown to select which search engine to use for queries (if enabled)
  • Search icon button: Appears when the search toolbar is docked vertically, providing a compact search interface with engine selector and input field
  • Suggestions: History, favorites, and remote suggestions appear as you type

When the search toolbar is docked to the left or right side of the window, the address bar and engine picker are replaced by a search icon button. Clicking this button opens a popup with the search engine selector and address input.

That covers all settings in Onu. Most changes take effect immediately. Some settings (like icon themes or Qt styles) may require restarting the browser for full effect. The configuration files are stored in plain text in ~/.config/Onu/Conf/onu.conf (public settings) and ~/.config/Onu/Data/encrypted.conf (encrypted data).
