2. Driver Manager - Keep Your Hardware Happy

This tool detects what hardware you have and helps install the right drivers.

What it detects:

  • Graphics Card (GPU) - Detects NVIDIA, AMD, or Intel graphics
  • Processor (CPU) - Identifies Intel or AMD processors

What you can do:

  • Install NVIDIA drivers - One click to get the proprietary NVIDIA driver for better gaming and 3D performance
  • Install printer drivers - Sets up CUPS printing system with support for most printers
  • Remove unused drivers - Safely clean up drivers for hardware you don't have (like AMD drivers if you have NVIDIA)

A terminal window will open for authentication - just enter your password when prompted, and the installation runs automatically.

4. App Remover - Clean Up Unwanted Software

This tool helps you uninstall applications you no longer need.

How to use: 1. Start typing the name of an installed application 2. Autocomplete will suggest matching packages 3. Select the app and click Remove

Important Note: Only remove applications you recognize. If you're unsure what a package does, search online first. Removing system-critical software could break your system.

A terminal window will open for confirmation and the removal process.

5. Wine Optimizer - Run Windows Software

Wine lets you run Windows applications on Linux. This tool helps you install, configure, and optimize Wine for the best experience.

Install / Update Tab: - Check your current Wine version - Install Wine Stable (recommended for most users) - Install Wine Staging (latest features, may be less stable) - Update existing Wine installation

Optimize Tab: - Gaming Preset - Enables ESYNC, FSYNC, installs DXVK, and optimizes registry settings for maximum gaming performance - Balanced Preset - Good for everyday use - Compatibility Preset - Better for older applications - Individual toggles for ESYNC, FSYNC, DXVK, VKD3D, and Large Address Aware

Configure Tab: - Create and manage Wine prefixes (separate Windows environments) - Set Windows version (Windows 10, 8.1, 7, or XP) - Launch winecfg (Wine configuration tool) - Launch regedit (Wine registry editor) - Launch taskmgr (Wine task manager)

Cleanup Tab: - Clear Wine cache to free disk space - Clean temporary files - Remove unused Wine prefixes

All changes happen through a terminal window where you can see progress and enter your password when needed.
